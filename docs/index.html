The problem

Roblox blocks Wine

The Hyperion anti cheat stops the Roblox Player from running under Wine. Native play was a dead end.

The runtime

Sober makes it work

Sober runs the Android Roblox client on Linux, the one path the anti-cheat allows. It just has no UI.

Pigment

The missing front end

Pigment drives Sober as it is and gives it everything Windows users get from Bloxstrap, and then some.

What you get

Everything, without the text editor.

Six focused tools over the Sober runtime. Each one reads and writes your real setup — safely.

Launcher

One-click Play

Launch Roblox through Sober, or make Pigment your roblox:// handler so browser “Play” buttons apply your setup first.

Settings

Every option, no JSON

A real UI over Sober’s config — graphics mode, Discord presence, gamemode, touch mode and more. Writes are atomic and validated.

FastFlags

Bloxstrap-compatible

Edit FastFlags as JSON in the exact format Bloxstrap uses. Paste a Windows preset to import, or clear them all in one click.

Mods

Validated against the APK

Install asset-overlay mods and Pigment checks every file against the real Roblox APK, flagging paths that won’t apply. Conflicts are detected too.

Profiles

Main, alt, testing

Save named setups of settings, flags and mods. The active profile is applied automatically each time you launch.

Activity

See what you played

Recent games parsed from Sober’s logs with names resolved live, plus opt-in Discord Rich Presence with elapsed time.

A look inside

Native GTK4, at home on your desktop.

Built with libadwaita. Light and dark, keyboard-friendly, and fast.

Install

Up and running in a minute.

You need the Sober Flatpak, plus GTK4 and libadwaita. Then pick your route.

1. Install the Sober runtime

# Sober is what actually runs Roblox
$ flatpak install flathub org.vinegarhq.Sober

2a. Per-user install (no root)

$ git clone https://github.com/SpeeNotPee/Pigment.git
$ cd Pigment
$ make install PREFIX=$HOME/.local
$ pigment

2b. Arch Linux (PKGBUILD)

$ make dist
$ cd packaging
$ makepkg -si

System-wide

$ sudo make install PREFIX=/usr

Then launch pigment from your app menu or terminal. Full setup and feature guides are in the documentation.
